Introduction to High Performance Computing

Using HPC at The University of Sheffield

This hands-on training course covers the subjects of;

  • What is High Performance Computing,
  • Managing data on the HPC facility,
  • Running interactive and batch jobs,
  • Accessing the available software.

Prerequisites

This course assumes that the students have a basic knowledge of Linux as covered by our Introduction to Linux course ( CIC6001a ). See also the Linux Cheatsheet.

Getting ready

Apply for an account on the HPC cluster (Iceberg) for details see the getting started information

Topics

  1. What is High Performance Computing
  2. HPC at The University of Sheffield
  3. Accessing HPC at TUOS
  4. Using Storage and Managing Data
  5. Software on the HPC Cluster
  6. Using the Job Scheduler
  7. GPUs and Remote Visualisation
  8. Beyond Iceberg
  9. Get Help and Information
